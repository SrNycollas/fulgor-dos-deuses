# Fulgor dos Deuses - Módulo para Foundry VTT

Este é um módulo de aventura para **Foundry Virtual Tabletop**, desenvolvido por fã, que adapta para o ambiente virtual a aventura oficial **Fulgor dos Deuses**, publicada pela **Jambô Editora** para **Tormenta20**.

O objetivo deste projeto é facilitar a preparação e condução da aventura dentro do Foundry VTT, oferecendo um compêndio organizado com os elementos necessários para a mesa.

# O que este módulo NÃO contém

## ❌ Este módulo NÃO disponibiliza o PDF da aventura.

O PDF oficial da aventura **não será distribuído**, hospedado ou disponibilizado de qualquer forma por este projeto.

Este módulo foi criado **apenas para auxiliar jogadores e mestres que já possuem legalmente a Revista Tormenta20 contendo a aventura**.

Caso você ainda não possua a aventura oficial, adquira-a pelos canais oficiais da **Jambô Editora**.

# Objetivo do projeto

Este módulo busca facilitar a utilização da aventura dentro do Foundry VTT, disponibilizando:

- Aventura pronta para importação;
- Estrutura organizada em compêndios;
- Cenas;
- Atores;
- Journals;
- Recursos utilizados durante a aventura.

Todo o conteúdo foi organizado pensando em agilizar a preparação do mestre durante a sessão.

---

# Requisitos

- Foundry Virtual Tabletop v13
- Sistema Tormenta20 compatível

---

# Instalação

No Foundry VTT acesse:

**Complementos → Instalar Módulo**

No campo **Manifest URL**, cole o seguinte endereço:

```text
https://raw.githubusercontent.com/SrNycollas/fulgor-dos-deuses/refs/heads/main/module.json
```

Depois clique em **Instalar**.

Ao finalizar:

1. Ative o módulo em seu mundo.
2. Abra o Compêndio **Fulgor dos Deuses**.
3. Importe a aventura normalmente.
